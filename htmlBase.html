              About

              pwManager is a clean, light and reliable password manager that is safe to use and works flawlessly. This password manager is just a project, and it is the first real website of mine.

              Since this website is in its early versions, there may be a few bugs and glitches. If you find any, please write an email to me, go to Contact if you want to find that email

              Short history

              pwManager was originaly created back in 2018, when I found out that enviroment variables existed on Windows and wondered if I could make a local password manager.

              So I started work on it as a python program, but eventually grew tired of it because I didn't find it interesting. I though of making a real website, at first it looked really bad,

              but as my skills expanded, I found out about 960.gs, eventually dumping it and instead used Bootstrap. I found out about jQuery, so making a website would be really easy. Here we are now,

              with this being presented at the Coolest Projects, I feel like I did really well in making this and it was a lot of fun!
